My Projects

Here there are some of the personal projects I've worked on:

Llm_calculator is a project that uses local llm zephyr to extract the information required for a basic operation (such as sum or multiply) from a prompt, to later perform the operation and returns the result to the user.
In EmotionRecognition I trained an image classifier to recognize the emotion that a person's face is showing. The inference is performed in real-time using the webcam.
RoomLeds is a project that implements several light patterns to be executed in a led strip, and allows you to controll it via a infrared remote controller.
